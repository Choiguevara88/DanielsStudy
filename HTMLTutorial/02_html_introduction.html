What is HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describs the structure of a Web page
  • HTML consists of a series of elements
  • HTML emelents tel the browser how to display the content
  • HTML elements label pieces of content such as "이것은 헤딩이다.", "이것은 파라그래프다.", "이것은 링크다.", etc
